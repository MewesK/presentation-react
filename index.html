React, what's that?

In a nutshell

  • "An open-source JavaScript library providing a view for data rendered as HTML."
  • Created by Jordan Walke in 2013
  • Currently maintained by Facebook and Instagram
  • Also used by Imgur, Airbnb and others
   

Just the UI

"Lots of people use React as the V in MVC. Since React makes no assumptions about the rest of your technology stack, it's easy to try it out on a small feature in an existing project."

facebook.github.io/react/

Virtual DOM

"React abstracts away the DOM from you, giving a simpler programming model and better performance. React can also render on the server using Node, and it can power native apps using React Native."

facebook.github.io/react/

Data Flow

"React implements one-way reactive data flow which reduces boilerplate and is easier to reason about than traditional data binding."

facebook.github.io/react/

Components

  • Components are the core concept of React
  • Used to encapsulate and modularize view components
  • Can be standalone or composed of other React components
  • Written in babel (a JavaScript dialect)

Example

var Hello = React.createClass({
  render: function() {
    ...
  }
});
ReactDOM.render(
  <Hello />,
  document.getElementById('content')
);

Rendering

  • React includes the templating language JSX
  • JSX is a XML-like syntax extension to ECMAScript
  • More info at facebook.github.io/jsx
  • JSX is recmmoneded but not required!

Example

var Hello = React.createClass({
  render: function() {
    return <div>Hello {this.props.name}</div>;
  }
});
ReactDOM.render(
  <Hello name="World" />,
  document.getElementById('content')
);

Data handling

  • React provides the internal structure for data handling.
    This includes:
  • Properties management (this.props)
  • State management (this.state)
  • Automatic re-rendering when the state changes (this.setState)
  • But it does not provide a dedicated way to communicate with a server!

Example

var Hello = React.createClass({
  load: function() {
    $.ajax({
      url: this.props.url,
      ...
      success: function(data) {
        this.setState({name: data});
      }.bind(this),
      ...
    });
  },
  getInitialState: function() {
    return {name: 'World'};
  },
  componentDidMount: function() {
    this.load();
    setInterval(this.load, this.props.pollInterval);
  },
  render: function() {
    return <div>Hello {this.state.name}</div>;
  }
});
ReactDOM.render(
  <Hello url="/api/name" pollInterval={2000} />,
  document.getElementById('content')
);

Pre-compiling

Install the Babel command-line tools (requires npm):

npm install --global babel-cli npm install babel-preset-react

Run the precompiler:

babel --presets react ''src'' --watch --out-dir ''build''
